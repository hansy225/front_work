* target속성 : 링크된 문서를 클릭했을 때 문서가 열릴 위치를 명시

_self(기본값) : 현재 열려있는 창에 링크가 열림
_blank : 새 탭에 링크창이 열림
_parent : 링크된 문서를 현재 프레임의 부모 프레임에서 오픈(프레임이 없으면 현재 탭)
_top : 링크된 문서를 현재 윈도우 전체에서 오픈함 (프레임이 없으면 현재 탭)
