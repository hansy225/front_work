폼 태그

form 태그 내에 submit버튼을 클릭 시 해당 form안의 작성된 내용
즉, 사용자 입력한 값들을 서버로 전송하면서 요펑하는 욕할 수행

<form action="요청을 처리할 페이지" method="" >

* action 속성 : 해당 form에 사용자가 입력한 값들을 전달하면서 요청할 서버의 경로
* method 속성 : 요청 전송 방식을 지정하는 속성 (get(기본값) | post)
> get : 요청 시 사용자가 입력한 값들이 url에 노출되는 방식
> post : 요청 시 사용자가 입력한 값들이 url에 노출되지 않는 방식
form내의 submit 버튼 클릭 시 action에서 지정한 서버로 사용자가 입력한 값들이 전송
사용자가 넣은 값은 항상 key=value의 세트로 전송
